# WanderNest

WanderNest is a smart accommodation platform that helps users explore verified stays, save favorite listings, post reviews, and access travel support services.

## Features

* User Authentication
* Add / Edit / Delete Listings
* Review System
* Interactive Map Integration
* Cloud Image Upload
* Wishlist Feature (planned)
* Vehicle Rental Module (planned)
* AI Complaint Assistant (planned)
* Dark / Light Mode (planned)

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* Bootstrap
* Cloudinary
* Mapbox
